    Creating Charges

    Use Assembly Payment’s API and your server-side code to process charges.

    Not a developer?

    Hire a Assembly Payments certified expert or use a pre-built solution created by one of our verified partners (no code required).

    After you securely collect and tokenize your customer’s credit card, you can charge the card immediately or save it for later.

    Unlike tokenization, which occurs in the browser, charge attempts are made from your server, normally using one of our client libraries. Install the library for your favorite language now, if you haven’t already done so.

    On your server, grab the Assembly Payments token in the POST parameters submitted by your form. From there, it’s one simple API call to charge the card:

    curl Ruby Python PHP Java Node Go .NET 
    // After you have securely tokenize your customer's credit card using AP's 
// tokenization service
  const chargeReq = {
    token: "4eC39HqLyjWDarjtT1zdp7dc",
    orderId: "order_1234",
    amount: 5000,
    currency: "AUD"",
    description: "Pizza"
  };
